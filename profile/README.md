# 🌟 Velkommen til Statens pensjonskasse (SPK) 🌟

Statens pensjonskasse (SPK) er Norges største leverandør av offentlige tjenestepensjoner. 
Vi sørger for at våre medlemmer får trygghet og økonomisk forutsigbarhet gjennom hele livet – fra første jobb til pensjonisttilværelsen.

## 🌍 Hva vi driver med
- Vi administrerer pensjonsrettigheter for rundt 1300 viksomheter og over én million medlemmer.
- Vi håndterer pensjoner for ansatte i staten, skoleverket og andre virksomheter med offentlig tilknytning.
- Vi tilbyr boliglån med konkurransedyktige betingelser for våre medlemmer.
- Vi utvikler og drifter moderne IT-systemer for å støtte effektive og pålitelige tjenester.

## 👩‍💻 Teknologi og innovasjon
Vi i SPK jobber kontinuerlig med å forbedre våre tjenester ved hjelp av moderne teknologi. 
Vårt teknologimiljø består av dyktige utviklere, arkitekter, designere, produkteiere og masse andre flotte folk.
